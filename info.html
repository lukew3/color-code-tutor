Color Code Tutor

This site was created to allow users to train their ability to identify colors as their digital representations

See more on how to read hex color codes here

For this site, your score on each submission is calculated by adding the absolute value of the differences of the guess minus the actual value. The lower the score, the better the guess.


If you're a developer, check out the source code here: https://github.com/lukew3/color-code-tutor
